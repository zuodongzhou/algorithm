### 题目描述

给定一个数组和滑动窗口的大小，找出所有滑动窗口里数值的最大值。例如，如果输入数组{2,3,4,2,6,2,5,1}及滑动窗口的大小3，那么一共存在6个滑动窗口，他们的最大值分别为{4,4,6,6,6,5}； 针对数组{2,3,4,2,6,2,5,1}的滑动窗口有以下6个： 

- {[2,3,4],2,6,2,5,1}
- {2,[3,4,2],6,2,5,1}
- {2,3,[4,2,6],2,5,1}
- {2,3,4,[2,6,2],5,1}
- {2,3,4,2,[6,2,5],1}
- {2,3,4,2,6,[2,5,1]}


### 思路

滑动窗口用队列保存，当遇到以下情况做相应处理：

- 1. 窗口第一个槽始终是最大值
- 2. 窗口第一个槽过期要丢弃
- 3. 窗口从尾比到头，丢弃较小的数
